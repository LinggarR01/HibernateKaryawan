# CRUD Mahasiswa Sederhana
Proyek aplikasi desktop sederhana untuk pengelolaan data mahasiswa, memanfaatkan teknologi modern untuk koneksi database dan pembuatan laporan.

## Fitur
- Mengelola data (Tambah, Baca, Ubah, Hapus) mahasiswa.
- Koneksi ORM Menggunakan Hibernate sebagai Object-Relational Mapping (ORM) untuk interaksi data yang terstruktur.
- Pembuatan laporan yang dapat dicetak menggunakan JasperReports.
- Export Data Cepat mahasiswa yang ditampilkan dalam tabel ke format PDF.

## Tech
- Netbeans IDE (8.2)
- MySQL Connector (8.0)
- Hibernate ORM
- XAMPP/Laragon
- JDK 8

## Setup
- Pastikan Anda telah menginstal JDK 8, NetBeans 8.2, dan XAMPP.
- Clone repositori ini
- Buka proyek di NetBeans IDE 8.2.
- Pastikan semua library (Hibernate, JasperReports, dll.) telah ditambahkan ke proyek Anda (klik kanan Libraries -> Add JAR/Folder).
- Klik tombol Run Project pada NetBeans (atau tekan F6).
